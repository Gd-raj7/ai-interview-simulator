# 🧠 AI Interview Simulator

An intelligent interview preparation system that evaluates user answers using NLP techniques such as TF-IDF similarity and keyword matching.

---

## 🚀 Features

* 🎯 Dynamic question generation (by category & difficulty)
* 🧠 NLP-based answer evaluation
* 🔍 Keyword extraction & matching
* 📊 Scoring system (Similarity + Keyword Score)
* 💡 Feedback with missing concepts

---

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn (TF-IDF, Cosine Similarity)
* Regex (Text Processing)

---

## ▶️ How to Run

```bash
cd src
python app.py
```

---

## 📊 Example Output

* Similarity Score: 0.68
* Keyword Score: 0.5
* Result: Partially Correct
* Missing Keywords: noise, patterns
